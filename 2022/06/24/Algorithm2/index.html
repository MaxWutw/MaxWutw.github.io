Modulo(模除)

There come’s an example, if you want to do below operate how will you do?

$5^{732} \pmod {682}$

You might want to calculate 5^732 first, then according to the result and do mod 682, although this method is executable, when we calculate, it might overflow, therefore we use the property of the modulo to solve it.
There is an algorithm about modulo:

  1. $(a+b)\mod m = ((a\mod m) + (b \mod m)) \mod m$
  2. $(a \cdot b)\mod m = ((a\mod m) + (b \mod m)) \mod m$

Exponentiating By Squaring(快速冪)

By using exponentiating by squaring this technique, we can get a faster execute time, and the time complexity is O(logN).

This algorithm turn the exponential to bits, for example: 10(Decimal) $\to$ 1010(Binary), if the bit is 1, and z multiplied by a, and we use shift operator to make our divide more faster, and we won’t stop this loop until b equals 0.
There we use $3^{5} \pmod {10007}$ to show how this algorithm work, in order to makes our algorithm easier, we will remove mod in below process, but remember to add mod in your code:

  1. Turn 5(Decimal) to binary is 101.
  2. The first bit is 1, therefore $z = (1 * 3)$.
  3. We do two times a is $3^2$ (9).
  4. Shift b and we and get the result 10.
  5. Because the next bit is 0, we don’t need to multiply z.
  6. We do two times a is $3^4$ (81).
  7. Shift b and we and get the result 1.
  8. The bit is 1, according to law of exponent we can multiply $3^4$ and $3^1$, and the radix is same hence we can get $z = (3^1 * 3^4)$, z equals $3^5 = 243$.

Fermat’s little theorem(費馬小定理)

THEOREM :

If there are two number a and p, and $gcd(a,p)=1$ it’s mean a and p are coprime, therefore below formula will established.

FORMULA:

$a^{p}\equiv a{\pmod {p}}$
The formula can be like this too:
$a^{p-1}\equiv 1{\pmod {p}}$

PROOF:

According to our condition $gcd(a,p)=1$, and a, 2a, 3a, …. ,(p-1)a mod p’s result just 1, 2, 3, …., (p-1) recombination.
Therefore:
$$a \cdot 2a \cdot 3a \cdot (p-1)a \equiv 1 \cdot 2 \cdot 3 \cdot (p-1) \mod p$$
$$ (p-1)!a^{p-1} \equiv (p-1)! \mod p$$
Then we divide $(p-1)!$ and we can get below formula.
$$ a^{p-1}\equiv 1{\pmod {p}} $$

Prefix Sum

1D

If we want to calculate the sum of an interval, we might want to use linear for loop to calculate the answer, but the solution is O(N), it might takes too long time to calculate, therefore we use prefix sum to solve this problem. Prefix sum time complexity is O(1), and prefix sum is a preprocess, after the preprocess, we can use a easy way to calculate the sum of an interval.

We construct two array, and the first one is storing our origin data, and the other is storing our prefix sum data, and we set our pre[0] = arr[0], then our prefix sum data is made by pre[i - 1] + arr[i].

After our preprocess, we can calculate the sum of an interval in O(N), if we want to calculate index 1 to 5’s sum, we can use pre[5] - pre[1 - 1], and we can get the answer.
